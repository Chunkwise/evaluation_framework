# Chunking evaluation (Chunkwise fork)

This repository is a fork of the **Chroma chunking evaluation framework**, adapted for use within **Chunkwise**.

## About

This fork includes refactors and fixes to better support Chunkwise evaluation workflows, including:

- Support for different chunker interfaces
- Improvements to evaluation handling (typing, corpus ID handling)
- Fix for collection initialization, and removal of unused code

For full documentation and background, refer to the original project:

- https://github.com/brandonstarxel/chunking_evaluation  
- https://research.trychroma.com/evaluating-chunking
